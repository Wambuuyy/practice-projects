# Vision Planner Project

## Project Overview
The Vision Planner is a web-based tool that allows users to:
- Define yearly goals
- Break down goals into actionable steps
- Upload vision images for each goal
- Create a fully customizable vision board
- Organize images and add text on the vision board
- Set board background color
- Print or export the vision board (future feature)
- Receive weekly reminders for their goals (future feature)

---

## Pages
1. **index.html** — Goals & Actions Page
   - Add goals dynamically
   - Add unlimited actions per goal
   - Upload images for each goal
   - Navigate to Vision Board page

2. **board.html** — Vision Board Page
   - Sidebar gallery with uploaded images from goals
   - Blank canvas for arranging images and text
   - Drag-and-drop images onto canvas
   - Resize images (future)
   - Add text boxes (future)
   - Set background color
